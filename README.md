# IdleBox
My shell tool kit.

## Installation

### with curl

```shell
zsh -c "$(curl -fsSL https://raw.githubusercontent.com/libook/IdleBox/master/install)"
```

### with wget

```shell
zsh -c "$(wget https://raw.githubusercontent.com/libook/IdleBox/master/install -O -)"
```
